Tentang Saya

The Person Behind the Code

Saya adalah mahasiswi semester akhir jurusan Teknik Informatika di STMIK Mardira Indonesia, yang sangat tertarik dengan titik temu antara logika, data, dan kecerdasan buatan. Melalui mata kuliah Kecerdasan Buatan (AI), saya tertantang untuk mengeksplorasi algoritma machine learning, strategi pencarian (search strategies), dan penyelesaian masalah komputasi dari dasar.

Portofolio ini merupakan rekam jejak digital dari perjalanan belajar saya — setiap tugas di sini adalah hasil dari proses riset, *debugging*, dan pemahaman mendalam. Saya percaya bahwa cara terbaik untuk memahami AI adalah dengan mempraktikkannya secara langsung, bukan sekadar teori.

Di luar jam kuliah, saya senang mengeksplorasi proyek *open-source*, membaca tren perkembangan teknologi AI terbaru, serta berkolaborasi dengan teman-teman untuk menyelesaikan berbagai tantangan teknis.
